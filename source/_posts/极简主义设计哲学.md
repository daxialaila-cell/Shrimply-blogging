> "Simplicity is the ultimate sophistication."
> — Leonardo da Vinci

## 什么是极简设计

极简不是"什么都没有"，而是**只留下必要的**。每一处留白都有意义，每一个元素都有理由存在。

![极简设计三原则](/img/posts/inline-design-principles.svg)

### 核心原则

**1. 减法是加法**

移除不必要的装饰，让内容成为主角。一个按钮不需要渐变、阴影、圆角全上——也许纯色 + 一个 hover 状态就够了。

**2. 间距即语言**

```
拥挤 ≠ 丰富
留白 ≠ 空洞
```

恰当的间距创造出"呼吸感"，让眼睛有休息的地方。

**3. 一致性 > 创意**

全站使用统一的：
- 字体层级（不超过 3 种字号）
- 色彩系统（主色 + 辅助色 + 中性色）
- 间距规则（4px / 8px 倍数体系）

**4. 内容驱动设计**

设计为内容服务，而不是反过来。先想清楚要表达什么，再决定怎么呈现。

### 实践中的极简

```css
/* 好：清晰明了 */
.card {
  padding: 24px;
  background: #fff;
  border-radius: 8px;
}

/* 不好：过度设计 */
.card {
  padding: 24px 28px;
  background: linear-gradient(135deg, #fff, #f8f9fa);
  border-radius: 12px 4px 16px 8px;
  box-shadow: 0 4px 20px rgba(0,0,0,0.08), 0 1px 3px rgba(0,0,0,0.04);
}
```

## 总结

极简是一种克制。知道**不做什么**，比知道做什么更重要。
